<h2>📖 About</h2>
<p>This is a solo project that I created to learn about APIs. It uses HTML, CSS, and JavaScript.</p>
<p>The website adjusts non-qualified players to be with other players. Last season, Jonathan Aranda batted .316, but only had 422 PA out of the minimum 502. In the case of batting average, 1 AB is added for every PA missed. Since Aranda had 50 non-PA's, the adjusted average uses 117 H and 450 AB, which would put him at a .260 AVG.</p>
<p>MLB's description of Tony Gwynn Rule (10.22(a)): <a href="https://www.mlb.com/news/some-mlb-rules-changes-no-big-deal-c217256694#:~:text=if%20a%20player%20did%20not%20have%20the%20qualifying%20number%20of%20plate%20appearances%2C%20he%20could%20still%20win%20the%20title%20if%20he%20still%20would%20have%20had%20the%20highest%20batting%20average%20if%20at%2Dbats%20were%20theoretically%20added%20to%20his%20total">"if a player did not have the qualifying number of plate appearances, he could still win the title if he still would have had the highest batting average if at-bats were theoretically added to his total".</a></p>
