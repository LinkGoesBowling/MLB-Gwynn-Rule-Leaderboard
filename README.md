<h3>📖 About</h2>
<p>This is a solo project that I created to learn about APIs. It uses HTML, CSS, and JavaScript.</p>
<p>The website adjusts non-qualified hitters to be with other players by adding hitless at-bats. A famous example of a player that fell short and won was Tony Gwynn in 1996. His original batting average was .353 but then it was adjusted to .349 since he fell short, and then he still won the batting title. I also programmed a rule for pitchers, although it is not based on an official rule that if a player falls short, 1 ER and 1 IP will be added for every missed inning.</p>
<p>MLB's description of Tony Gwynn Rule (10.22(a)): <a href="https://www.mlb.com/news/some-mlb-rules-changes-no-big-deal-c217256694#:~:text=if%20a%20player%20did%20not%20have%20the%20qualifying%20number%20of%20plate%20appearances%2C%20he%20could%20still%20win%20the%20title%20if%20he%20still%20would%20have%20had%20the%20highest%20batting%20average%20if%20at%2Dbats%20were%20theoretically%20added%20to%20his%20total">"if a player did not have the qualifying number of plate appearances, he could still win the title if he still would have had the highest batting average if at-bats were theoretically added to his total".</a></p>
<h3>✨ Features</h2>
<ul>
  <li>Lists all qualified players with their normal averages</li>
  <li>Calculates adjusted averages for non-qualified players</li>
  <li>Lists qualified and adjusted players together</li>
  <li>Highlights non-qualified players in red if checkmark is boxed</li>
</ul>
